# IPL-ANALYSIS
IPL 2022 Analytics Dashboard | Power BI
Project Description:
Developed an interactive IPL 2022 Analytics Dashboard using Power BI to analyze match, batting, bowling, toss, venue and player performance throughout the season. The dashboard provides actionable insights through dynamic visualizations, KPI cards, filters, and drill-down analysis.
Key Highlights:
Analyzed 74 IPL 2022 matches and season-wide performance metrics.
Built dashboards for Match Analysis, Toss Analysis, Batting Performance, Bowling Performance, and Player of the Match Analysis.
Identified top run scorers, highest wicket takers, best bowling figures, and most Player of the Match award winners.
Analyzed toss decisions, win patterns, innings-wise scoring trends, and score distributions.
Designed interactive navigation, slicers, KPI cards, bar charts, donut charts, and distribution visualizations.
Applied DAX calculations and data modeling techniques for performance analysis.
Tools & Technologies:
Power BI
DAX
Excel/CSV
Data Modeling
Data Visualization
Business Value: The dashboard enables stakeholders to quickly explore team and player performance, uncover match-winning trends, and gain insights into batting, bowling, and toss strategies during IPL 2022.
